# 更新日誌

此文件記錄專案的版本歷史和重要變更。

## v2.0.0 (2025-01-08)

**重大重構：專案結構改為按遊戲分類**

### 新功能
- 新增 `/zenless-zone-zero` 指令（絕區零兌換碼查詢）

### 重構
- 專案結構改為按遊戲分類（`src/games/hsr/`, `src/games/zzz/`）
- Python 設定檔（`pyproject.toml`, `uv.lock`）移至專案根目錄
- HSR 相關檔案從 `src/services/` 和 `src/bot/commands/` 移至 `src/games/hsr/`

### 改進
- 兌換碼查詢改用 JSON 格式解析，提高穩定性
- 新增 NDJSON 解析支援
- 兌換碼加入可點擊連結
- AI 回應加入「AI 可能會出錯」提示
- 限制兌換碼搜尋範圍為 7 天內
